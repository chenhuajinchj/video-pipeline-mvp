两种接入方式

REST API

标准 HTTP 接口，支持 curl、Python、任何 HTTP 客户端调用。实时 SSE 进度推送。

查看 API 文档 →

MCP Server

Model Context Protocol 原生支持，Claude Desktop / Claude Code 直接调用，AI-to-AI 无缝协作。

API 速览

POST/api/storyboard — 输入逐字稿，生成分镜表
POST/api/images/{id} — 批量生成图片（SSE 流）
GET/api/projects — 列出所有项目
GET/api/projects/{id} — 项目详情
PUT/api/projects/{id}/storyboard — 编辑分镜
GET/api/download/{id} — 打包下载

快速开始

# 1. 生成分镜
curl -X POST http://localhost:8600/api/storyboard \
  -H "Content-Type: application/json" \
  -d '{"script_text": "你的逐字稿内容...", "style": "AI科技"}'

# 2. 生成图片（返回 SSE 流）
curl http://localhost:8600/api/images/{project_id} \
  -X POST -H "Content-Type: application/json" \
  -d '{"style": "tech", "concurrency": 3}'

# 3. 下载项目
curl -O http://localhost:8600/api/download/{project_id}

MCP 配置

在 Claude Desktop 的 claude_desktop_config.json 中添加：

{
  "mcpServers": {
    "video-pipeline": {
      "command": "python",
      "args": ["-m", "mcp_server.server"],
      "cwd": "/path/to/video-pipeline-mvp",
      "env": {
        "GEMINI_API_KEY": "your-api-key"
      }
    }
  }
}
